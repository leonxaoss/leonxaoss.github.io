Модуль Lookbase v1.0

Вимоги:

Версія DLE: 10.2 та вище

Протестовано в версіях DLE від 10.2 до 13.3

Підтримувана кодування: UTF-8

Версія php: 5.4 та вище

Підтримувана кодування: UTF-8 + cp1251

Що вміє:

  • 1. Ручний пошук і вставка посилання на відео / серіал
  • 2. Автоматична постановка посилання на відео / серіал через розділ "Масове проставлення посилань"
  • 3. Автоматичне проставляння / оновлення посилань по cron

Завантажити: lookbase_v1.0_2020_02_20.zip (522 кб)

Установка автоматична. Але є і інструкція для ручної установки

JS-Скрипт додавання плеєру за id kinopoisk

Що вміє:

  • 1. Установка плеєра на сторінці при наявності id kinopoiska

Інструкція:

  • 1. Візьміть код і встановіть на сторінки сайту: code.txt
  • 2. Створіть контейнер (наприклад div), в якому буде розгорнуто плеєр
  • 3. Налаштуйте скрипт

Налаштування:

forElem : "#invideo", // селектор елемента (напр.div) для інтеграції відео;

inName : "idkp", // ім'я GET параметра містить kp id (& idk = 000000000);

inValue : "5453232", // або пряма вказівка kp id (більший пріоритет);

token : "xxx", // api токен запитувача користувача;

width : "740px", // ширина фрейму відео, інакше 100% ширини батька (forElem);

height : "480px" // висота фрейму відео, інакше 100% висоти батька (forElem);

API

Для запитів вам буде потрібно Ваш API key:

m23-p0GG-pA-C

Отримати свій API key ви можете в особистому кабінеті користувача.

Запит слід відправляти на:

https://lookbase.tv/api/

Тип запиту: GET

Приймаються наступні параметри:

id_kp > ID відео kinopoisk, приклад 1023065

id_imdb > ID відео Internet Movie Database (IMDb), приклад tt6464360

original_name > оригінальна назва фільму, приклад A Rainy Day in New York

token > ваш API key

Пошук здійснюється по одному з параметрів, в послідовності id_kp> id_imdb> original_name до першого входження,

таким чином, при виявленні фільму відповідного id_kp,

подальший пошук або перевірка відповідності знайденого фільму з id_imdb, original_name - не виробляються!

Приймаються наступні параметри:

https://lookbase.tv/api/?id_kp=1023065&id_imdb=tt71339936&original_name=A%20Rainy%20Day%20in%20New%20York&token=m23-ApGnnC2-C

Приклад запиту тільки з id_imdb:

https://lookbase.tv/api/?id_imdb=tt7139936&token=m23-ApGnnC2-C

Формат відповіді: JSON

Ключ status - може приймати значення error, або success

Структура відповіді з помилкою (status == error):

{
"status":"error",
"message":"the search has not given any results"
}

У цьому випадку ключ message буде містити опис помилки;

Структура відповіді з успішним пошуком (status == success):

В цьому випадку в ключі data міститиметься колекція даних про знайдений фільм:

{
"status":"success",
"data":
  {
  "id_kp":"52",
  "id_imdb":"tt7139936",
  "ua_name":"Дождливый день в Нью-Йорке",
  "orig_name":"A Rainy Day in New York",
  "link":"https://lookbase.xyz/sc/m23-Otk/m23-LG2/m23-nCLpp/"
  }
}

Де:

id_kp > ID відео kinopoisk, приклад 1023065

id_kp > ID відео kinopoisk

id_imdb > ID відео Internet Movie Database (IMDb)

Id_local > (int) локальний ID в нашій базіua_name > (string) назва українською мовою

original_name > (string) оригінальна назва фільму
